# 3Dファイルビューアー（GENEROSITY技術課題）

 GENEROSITYの技術課題のGLB/GLTFモデルビューアーです。
 
 [こちら](https://dylanburns-net.github.io/generosity-coding-test/)でGithub Pagesでオンラインでご確認いただけます。
 
 
## 操作方法
| | 回転 | 並行移動 | 拡大/縮小 |
| ------------- | ------------- | ------------- | ------------- |
| マウス | 左ボタン | 右ボタン | 中ボタン |
| キーボード | Shift + WASD | WASD | 矢印キーの上下  |
